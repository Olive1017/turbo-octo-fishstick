# Shell Print Project

## Overview
The Shell Print Project is an automatic printing tool developed in Python that features a user-friendly graphical user interface (GUI). This project aims to streamline the printing process, making it accessible and efficient for users.

## Features
- **Automatic Printing**: Automatically send documents to the printer without manual intervention.
- **User-Friendly GUI**: Intuitive graphical interface for easy navigation and operation.
- **Multi-Format Support**: Supports various document formats for printing, including PDF, DOCX, and images.
- **Customizable Settings**: Users can configure printer settings based on their preferences.

## Installation
To get started with the Shell Print Project, follow the instructions below:

### Requirements
- Python 3.6 or higher
- Tkinter library (comes pre-installed with Python)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Olive1017/turbo-octo-fishstick.git
   ```
2. Navigate to the project directory:
   ```bash
   cd turbo-octo-fishstick
   ```
3. Run the main application:
   ```bash
   python main.py
   ```

## Usage
1. Launch the application by running the main script.
2. Load the document you wish to print.
3. Adjust the printer settings if necessary and hit 'Print'.
